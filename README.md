Sampling
========

Explore different methods of generating a small random subset of
a population.

 + S0: first blood
 + S1: using bitsets
 + S2: sparse sets
 + SFabrice: josephus sampling

Compile:
--------

> ocamlbuild -use-ocamlfind benchmark.native

Run:
----

> ./benchmark.native --module S0
